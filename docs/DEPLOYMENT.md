# デプロイメント

## デプロイ手順

1.  ビルドを行う。
    ```bash
    npm run build
    ```
2.  ビルドされたファイルをデプロイする。
    *   例: Netlify, Vercel, AWS S3など
3.  環境変数を設定する。
    *   デプロイ環境に合わせて環境変数を設定する。
    *   例: `API_URL`, `DATABASE_URL`など
