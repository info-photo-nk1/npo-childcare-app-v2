## インストール方法

1.  プロジェクトをクローンする。
    ```bash
    git clone <リポジトリURL>
    cd <プロジェクトディレクトリ>
    ```
2.  依存関係をインストールする。
    ```bash
    npm install
    ```
3.  開発サーバーを起動する。
    ```bash
    npm run dev
    ```
4.  ブラウザで `http://localhost:5173` にアクセスする。
